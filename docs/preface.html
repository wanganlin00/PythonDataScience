在文中使用的集成开发环境（IDE）是Rstudio，code 表示代码（code）

R代码块中， #> 表示运行R代码输出的结果，如

Code
print("R")
#> [1] "R"

Python代码块中， #>>> 表示运行Python代码输出的结果，如

Code
print("Python")
#>>> Python

在 Rstudio Console 中交互式运行 Python

  • reticulate::repl_python () 进入 Python 环境

  • exit / quit 退出 Python 环境

Code
library(reticulate)
py_config()

reticulate::repl_python()

# 前言 {.unnumbered}

在文中使用的集成开发环境（IDE）是Rstudio，`code` 表示代码（code）

R代码块中， `#>` 表示运行R代码输出的结果，如

```{r}
#| comment: "#>"
print("R")
```

Python代码块中， `#>>>` 表示运行Python代码输出的结果，如

```{python}
print("Python")
```

## **在 R**studio **Console 中交互式运行 Python**

-   `reticulate::repl_python ()` 进入 Python 环境

-   `exit / quit` 退出 Python 环境

```{r}
#| comment: "#>"
#| eval: false
library(reticulate)
py_config()

reticulate::repl_python()
```
